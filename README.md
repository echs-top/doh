# doh-hosts

部署在vercel/netlify([enhanced-FaaS-in-China](https://github.com/xingpingcn/enhanced-FaaS-in-China))，只支持查询hosts中有的域名(action自动更新)

本项目的初衷为通过doh解决mihomo中fcm等域名的直连问题，比如我自己的项目:[模板](https://github.com/echs-top/proxy)，此项目的生命周期将终结于mihomo支持拉取hosts规则之时
